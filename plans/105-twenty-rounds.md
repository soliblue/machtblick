### Bugs (fix worthy, worst first) — all 4 FIXED by frontend, round 6 fixes 2026-07-04

1. **No visible keyboard focus on the /members search input.** Route /members (de + en), all viewports. Repro: click or Tab into the search field, nothing changes visually. Computed style while focused: `outline: none`, `box-shadow: none`, border stays at 15% hairline. Cause: the input combines the `outline-none` class with `focus:border-fg`, but the inline `style={{ borderColor: HAIR }}` always wins over the focus class, so `focus:border-fg` can never apply. Suspected file: `apps/bundestag/src/views/membersList/MembersList.tsx` (input around lines 88-96). Card links are fine (default `outline: auto 1px` survives).
2. **FilterSheet is not modal despite `role=dialog aria-modal=true`.** Routes /members and /votes (shared component), mobile only, both locales. Repro at 390px: open the floating Filter sheet. (a) Escape does not close it. (b) Focus stays on the Filter button behind the overlay when the sheet opens (activeElement outside the dialog). (c) No focus trap, Tab walks into background content. (d) Background page is not scroll-locked while the sheet is open. Backdrop tap and swipe-down work. Suspected file: `apps/bundestag/src/views/votesList/FilterSheet.tsx`.
3. **FilterPill dropdown and SortControl listbox ignore Escape.** Route /members desktop (also /votes pills), both locales. Repro at 1440px: open any filter pill (e.g. Fraktionen), press Escape, listbox stays open; only an outside mousedown closes it. Same pattern in the sort dropdown. Suspected files: `apps/bundestag/src/views/votesList/FilterPill.tsx`, `apps/bundestag/src/views/membersList/SortControl.tsx` (both register only a `mousedown` close listener).
4. **Filter label inconsistency between idioms (copy nit).** Desktop pill row labels the party filter "Fraktionen" (`t.navParties`) while the mobile sheet labels the same group "Fraktion" (`t.parliamentaryGroup`); in en it is "Parties" vs a bare "Group", and "Group" standing alone in the sheet is unclear. Suspected file: `apps/bundestag/src/views/membersList/MembersList.tsx` (sheetGroups line 65 vs FilterPill line 102).
